# KalturaPlayer

## Getting Started

To integrate KalturaPlayer into your Xcode project, specify it in your `Podfile`:

```ruby
pod 'KalturaPlayer'
```


## Samples

The samples can be found in the [KalturaPlayer Samples repo](https://github.com/kaltura/kaltura-player-ios-samples).
